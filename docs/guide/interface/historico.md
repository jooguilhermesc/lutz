# Histórico de Análises

O **drawer de Histórico** é acessado pelo botão **Histórico** na barra superior. Ele exibe as últimas execuções de análise do projeto sem sair da tela atual.

![History Drawer](/screenshots/history.png)

---

## O que é exibido

Cada card de execução mostra:

| Campo | Descrição |
|---|---|
| **Data e hora** | Timestamp da análise (ex: `14/07/2026, 09:43:22`) |
| **Artigos** | Quantidade de artigos processados |
| **Identificador** | Nome do arquivo de relatório |
| **Modelo** | LLM utilizado na análise |
| **Tokens** | Total de tokens consumidos |
| **Duração** | Tempo de execução |

---

## Ver todos os relatórios

Clique em **Ver todos →** para navegar à aba Relatórios, onde é possível abrir, visualizar e baixar cada relatório em PDF.

---

## Fechar o drawer

Clique no **×** ou em qualquer área fora do drawer para fechá-lo.
